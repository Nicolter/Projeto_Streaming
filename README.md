# Nicolter_repos

🎬 Sistema de Streaming (Projeto Web)

Projeto desenvolvido com foco em simular uma plataforma de streaming, utilizando tecnologias web e integração com banco de dados.

💻 Tecnologias utilizadas
PHP
JavaScript
HTML
CSS
Banco de dados (SQLServer)
⚙️ Funcionalidades
Cadastro de usuários
Login de usuários
Exibição de conteúdos
Integração com banco de dados
👥 Desenvolvimento

Projeto desenvolvido em equipe com utilização de programação em dupla (pair programming).

Atuamos de forma colaborativa no desenvolvimento full stack da aplicação, participando tanto do front-end quanto do back-end.
Fui responsável, em conjunto com outro integrante, pela implementação da lógica do sistema, integração com banco de dados, sistema de login, funcionalidades de interação (like/dislike) e recursos como sistema premium.
Também participei da estruturação do projeto e da estilização (CSS).

🎯 Objetivo

Aplicar conhecimentos de desenvolvimento web, lógica de programação e integração entre front-end e back-end.
